# Logistics Tracking Platform

A production-ready Java Backend Microservices project that simulates a real-world logistics tracking system.

## Tech Stack

- Java 21
- Spring Boot
- Spring Security
- JWT Authentication
- Spring Data JPA
- Hibernate
- MySQL
- Redis
- Apache Kafka
- Docker & Docker Compose

## Microservices

- Authentication Service
- Shipment Service
- API Gateway
